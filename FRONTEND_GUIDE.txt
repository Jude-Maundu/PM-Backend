# FRONTEND FEATURES & API INTEGRATION GUIDE

## 1. AUTHENTICATION
**Location:** /pages/auth or /components/Auth

### Features:
- [ ] Register Page
  - API: POST /api/auth/register
  - Fields: username, email, password, profilePicture, role (user/photographer/admin)
  - Response: user object with ID

- [ ] Login Page
  - API: POST /api/auth/login
  - Fields: email, password
  - Response: user object with token (implement JWT)
  - Store user in localStorage/context

- [ ] Auth Context/State Management
  - Store user info globally
  - Handle logout (clear localStorage)


## 2. HOME/BROWSE PHOTOS
**Location:** /pages/Home or /components/MediaGallery

### Features:
- [ ] Display All Photos Grid
  - API: GET /api/media
  - Show: title, thumbnail, photographer name, price
  - Add search/filter functionality
  
- [ ] Photo Search
  - Search by: title, type (photo/video), price range
  - API: GET /api/media/search?title=...&mediaType=...&minPrice=...&maxPrice=...

- [ ] Photo Details Modal/Page
  - API: GET /api/media/:id
  - Show: full image, description, photographer profile, price
  - Buttons: "Add to Cart", "Buy Now"

- [ ] Right-Click Protection (Frontend)
  - Disable right-click on images: e.addEventListener('contextmenu', e.preventDefault())
  - Add watermark overlay on images
  - Disable drag-drop of images


## 3. SHOPPING CART
**Location:** /pages/Cart or /components/Cart

### Features:
- [ ] View Cart
  - API: GET /api/payments/cart/:userId
  - Display: all items, quantities, total price
  - Buttons: Remove item, Clear cart, Checkout

- [ ] Add to Cart
  - API: POST /api/payments/cart/add
  - Body: {userId, mediaId}
  - Show success toast notification

- [ ] Remove from Cart
  - API: POST /api/payments/cart/remove
  - Body: {userId, mediaId}
  - Update cart display

- [ ] Clear Cart
  - API: DELETE /api/payments/cart/:userId
  - Confirmation dialog before clearing

- [ ] Cart Icon Badge
  - Show item count in navigation


## 4. PAYMENT & CHECKOUT

### Features:

## 5. RECEIPTS & INVOICES
**Location:** /pages/Receipt or /components/Receipt

### Features:
- [ ] View Receipt
  - API: GET /api/payments/receipt/:receiptId
  - Show: receipt number, date, items, total, breakdown

- [ ] User Receipts List
  - API: GET /api/payments/receipts/:userId
  - Display all past receipts

## 6. USER PURCHASE HISTORY
**Location:** /pages/MyPurchases or /pages/Dashboard

### Features:
- [ ] View Purchase History
  - API: GET /api/payments/purchase-history/:userId
  - Show: photos purchased, dates, prices
  - Status: completed, refunded

- [ ] Download Purchased Media
  - API: GET /api/payments/purchase-history/:userId (returns signed URLs)
  - Button: Download with signed URL protection
  - Expiration: signed URLs expire after 1 hour

- [ ] Filter/Sort Purchases
  - By date, price, photographer
  - Search by photo name


## 7. REFUND MANAGEMENT
**Location:** /pages/MyPurchases or /pages/Account

### Features:
- [ ] Request Refund
  - API: POST /api/payments/refund/request
  - Body: {paymentId, reason}
  - Show: confirmation message

- [ ] View Refund Status
  - API: GET /api/payments/refunds/:userId
  - Show: all refund requests with status
  - Status: pending, approved, rejected, processed

- [ ] Refund Details
  - Show reason, amount, status
  - If rejected: show admin's rejection reason
  - If processed: confirmation of wallet credit

  - Show: current balance, total earned, total withdrawn

- [ ] Wallet History/Transactions

## 9. PHOTOGRAPHER DASHBOARD
**Location:** /pages/PhotographerDashboard or /dashboard/photographer

### Features:
- [ ] Earnings Summary
  - API: GET /api/payments/earnings-summary/:photographerId
  - Show: total earned, current balance, recent sales
  - Display: top selling photo, average price

- [ ] Upload/Manage Photos
  - API: POST /api/media (create)
  - Form: title, description, price, category, file upload
  - Show uploaded photos list

- [ ] Set Price Per Photo
  - API: PUT /api/media/:id/price
  - Body: {price, photographerId}
  - Show: current price, ability to edit
  - Confirmation before saving

- [ ] View Sales
  - API: GET /api/payments/earnings/:photographerId
  - Table: photo name, buyer, date, amount
  - Export sales report (CSV)

- [ ] Edit Media Details
  - API: PUT /api/media/:id
  - Update: title, description, category
  - Delete photo: DELETE /api/media/:id

## 12. PROTECTED MEDIA ACCESS

- [ ] Generate Secure Download Link
  - API: GET /api/media/:id/protected (requires auth)

- [ ] Download with Token Verification
